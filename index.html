Moxa 11x0 USB Serial Driver by customink

This is an updated driver for the Moxa USB to Serial 11x0 family adapters that runs on Linux kernel 2.6.32 (x86 or x64).

This is an updated Moxa USB-to-RS232 U1100 series driver that will run on the Linux 2.6.32 kernel (x86 or x64). Moxa's last driver update was for the Linux 2.6.27 kernel.

Dependencies

Linux Kernel 2.6.32, Ruby 1.9.2 for tests
